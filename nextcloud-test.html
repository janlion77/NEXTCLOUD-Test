Nextcloud-Backup Test

Mini-App zum Prüfen, ob Faultiers Backup-Mechanik mit deiner Schul-Nextcloud zusammenspielt.

1. Zugangsdaten

Beim ersten Test gerne mit dem normalen Passwort starten, später durch ein App-Passwort ersetzen (in Nextcloud unter Einstellungen → Sicherheit → App-Passwort erstellen).

2. Tests

Protokoll

Was teste ich hier eigentlich?

① Login: PROPFIND auf das Benutzerverzeichnis. Antwortet die Nextcloud mit 207 Multi-Status, sind Server, URL und Zugangsdaten in Ordnung.

② Ordner: MKCOL auf den Backup-Ordner. 201 Created = neu angelegt, 405 Method Not Allowed = existiert bereits (auch okay).

③ Upload: PUT einer kleinen JSON-Datei mit Zeitstempel. Erwartet wird 201 Created oder 204 No Content.

④ Download: GET derselben Datei und Byte-für-Byte-Vergleich, damit klar ist, dass nichts unterwegs verändert wurde.
